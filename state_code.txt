# 状态码

0    业务正常完成

-1   用户业务错误
-11  用户登录错误
  -111  登陆失败，用户名或密码错误
  -112  登陆失败，用户不存在
-12  用户注册错误
  -121  注册所填用户名已被使用！
  -122  注册失败
  -123  邀请码错误，注册失败
-13   用户信息更改错误
  -131  旧密码输入错误

-2   云盘业务错误
-21  管理错误
  -211 文件检索错误
  -212 文件上传有同名文件
  -213 文件名为空
  -214 部分文件移动失败
  -215 全部文件移动失败
  -216 文件移动参数错误
-22  权限错误
  -221 没有访问权限
-23  文件分享错误
  -231 文件分享失败, 服务器错误
  -232 文件分享失败，所分享的文件不存在
-4   服务器错误
